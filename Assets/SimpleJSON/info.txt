пример упаковки серилиазированного класса в json

класс Json в папке Utills имеет три метода:
    IsExistsSave(path) проверяет есть ли файл-сохранения по указанному пути
    Load(path) загрузит файл-сохранения по указанному пути
    Save(path, data) сохранит данные по указанному пути

json в отличии от BinarySerialization может упаковывать и Vector3

в примере показана реализация простого текстового инвентаря
