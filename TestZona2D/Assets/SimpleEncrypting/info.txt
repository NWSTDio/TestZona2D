Пример работы с шифровкой/расшифровкой данных

Статический класс EncryptingUtills имеет несколько методов шифровки/расшифровки данных
    1. кодирование и разкодирование в base64 (небезопасно, любой через онлайн-сервис может расшифровать)
        string Encode64(value) принимает строку для шифровки, возвратит шифрованную строку
        string Decode64(value) принимает строку зашифрованную через Encode64() для расшифровки, возратит расшифрованную строку если это возможно
    2. перевод из строк в массив байтов и обратно (шуточный шифровщик), больше нужен как системная утилита для шифровщиков
        byte[] GetBytesFromString(value) получает строку, и возвратит массив байтов
        string GetStringFromBytes(value[]) получает массив байтов, не обязательно полученный через GetBytesFromString, и возвратит строку
    3. быстрая шифровка (более-менее надежно), подходит для шифровки ключей))
        3.1 с системным ключом
            string FastEncrypt(value) шифрует строку, с использованием системного ключа
            string FastDecrypt(value) разшифрует строку, зашифрованную выше с использованием системного ключа
        3.2 с пользовательским ключом
            string FastEncrypt(value, key) шифрует строку, с использованием пользовательского ключа
            string FastDecrypt(value, key) разшифрует строку, зашифрованную выше с использованием пользовательского ключа, совпадающего с ключем выше
    4. шифрока по AES (надежно)
        шифровка возможна двумя способами:
            string Encrypt(value, password) - принимает строку и пользовательский пароль, возвратит шифрованную строку
            string Encrypt(value[], password) - принимает массив байт и пользовательский пароль, возвратит шифрованную строку
        расшифровка:
            string Decrypt(value, password) - принимает зашифрованную строку выше, и пароль использованный выше, возвратит расшифрованную строку если пароль для шифрованной строки совпал
