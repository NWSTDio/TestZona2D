отработка ситуации когда Игровой обьект находится под UI элементом и происходит событие клика по кнопке

1. создать пустой обьект
    1.1 добавить Box Colider 2D
    1.2 добавить скрипт схожий со скриптом Block.cs
        OnMouseDown - метод события клика по обьекту
2. создать UI кнопку
    2.1 добавить скрипт схожий со скриптом ButtonUI.cs

*** в методе OnMouseDown для любого игрового обьекта перед обработкой логики клика вставить проверку:
    if (Functions.IsClickUIObject()) // если был клик по UI
        return;

- Класс Vector2Extensions содержит метод который расширяет структуру Vector2

- Класс Functions содержит методы:
    IsClickUIObject() для проверки находится ли над игровым обьектом UI элемент
    GetRandomColor(float alpha) возвратит рандомный цвет с сохранением альфы
